# BMI Calculator

BMI Calculator is a simple and interactive web application that calculates the Body Mass Index (BMI) based on user-provided weight and height.  
It helps users quickly check whether they are underweight, normal, overweight, or obese.  
Built with **HTML**, **CSS**, and **JavaScript**.

---

## 🌐 Live Demo

Try it here: [https://bahadirbektore.github.io/bmi-calculator/](https://bahadirbektore.github.io/bmi-calculator/)

---

## 🚀 How to Use

1. Enter your weight in kilograms (kg).  
2. Enter your height in meters (m).  
3. Click the **"Calculate BMI"** button.  
4. The result will show your BMI value and the corresponding category:  
   - **Underweight**  
   - **Normal weight**  
   - **Overweight**  
   - **Obese**
